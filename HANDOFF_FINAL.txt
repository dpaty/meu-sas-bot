🔒 GARANTIA: BOT PARA (3 Verificações)
═══════════════════════════════════════════════════════════════

1️⃣  ENTRADA
    ConversationController.avaliarConversa()
    └─ Se silenciado → retorna false
    └─ handleIncomingMessage() → return (não processa)

2️⃣  MEIO (Double-Check)
    Antes de chamar aiService
    └─ if (silencio.estaSilenciado(chatId)) return

3️⃣  SAÍDA (Triple-Check)
    Antes de enviar resposta
    └─ if (silencio.estaSilenciado(chatId)) return (não envia)

🚀 COMO USAR
═══════════════════════════════════════════════════════════════

OPÇÃO 1: VIA CURL (Teste Rápido)
──────────────────────────────

# Assumir atendimento
curl -X POST http://localhost:3000/api/handoff/silenciar/user123 \
  -H "Content-Type: application/json" \
  -d '{"chatId":"5521987654321@c.us"}'

# Liberar bot
curl -X POST http://localhost:3000/api/handoff/desilenciar/user123 \
  -H "Content-Type: application/json" \
  -d '{"chatId":"5521987654321@c.us"}'


OPÇÃO 2: NO DASHBOARD (Recomendado)
────────────────────────────────

<script src="handoff-integrator.js"></script>

<script>
    const handoff = new HandoffIntegrator('seu_user_id');
    
    // Assumir (ativa quando operador clica botão)
    await handoff.assumir('5521987654321@c.us');
    
    // Liberar (desativa quando operador termina)
    await handoff.liberar('5521987654321@c.us');
    
    // Atualizar UI
    setInterval(() => handoff.atualizarUI(), 10000);
</script>

✅ ENDPOINTS DISPONÍVEIS
═══════════════════════════════════════════════════════════════

GET  /api/handoff/silenciados/:userId
     Retorna lista de chats em atendimento humano

POST /api/handoff/silenciar/:userId
     Body: {"chatId": "5521987654321@c.us"}
     Ação: Operador assume → Bot silencia

POST /api/handoff/desilenciar/:userId
     Body: {"chatId": "5521987654321@c.us"}
     Ação: Libera bot → Retoma automático

GET  /api/handoff/status/:userId/:chatId
     Retorna: {estaSilenciado, status, silenciadoEm}

GET  /api/handoff/estado/:userId
     Retorna estado geral de atendimentos

🎨 INTEGRAÇÃO NO DASHBOARD (Exemplo Completo)
═══════════════════════════════════════════════════════════════

<!-- Adicione no seu dashboard-v2.html -->

<script src="handoff-integrator.js"></script>

<table class="table">
  <thead>
    <tr>
      <th>Cliente</th>
      <th>Status</th>
      <th>Ação</th>
    </tr>
  </thead>
  <tbody id="chats-lista">
    <!-- Preenchido dinamicamente -->
  </tbody>
</table>

<script>
  const handoff = new HandoffIntegrator('seu_user_id');
  
  // Recarregar lista a cada 10 segundos
  setInterval(async () => {
    const atendimentos = await handoff.listarAtendimentos();
    
    document.getElementById('chats-lista').innerHTML = atendimentos
      .map(chat => `
        <tr data-chat-id="${chat.chatId}">
          <td>${chat.chatId}</td>
          <td>
            <span class="badge badge-warning">Humano Atendendo</span>
          </td>
          <td>
            ${handoff.gerarBotao(chat.chatId, true)}
          </td>
        </tr>
      `).join('');
  }, 10000);
</script>
