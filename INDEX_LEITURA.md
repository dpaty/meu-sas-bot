## ⚡ TROUBLESHOOTING RÁPIDO

**Erro: "MASTER_GEMINI_KEY não encontrada"**
```bash
# Solução: Verificar .env
echo $MASTER_GEMINI_KEY
# Se vazio, adicionar: MASTER_GEMINI_KEY=gsk_seu_token
```

**Erro: "incrementMessageCounter não é função"**
```bash
# Solução: Restart do server
npm start
# Force reload:npx nodemon src/server.js
```

**Cliente não é criado**
```bash
# Solução: Verificar ID único
sqlite3 database.sqlite
SELECT id FROM users;
# O ID precisa ser único!
```

## 💬 FAQ DO PROJETO

**P: Posso usar outra IA além de Gemini?**
R: Sim, está preparado para isso. Mude em `src/services/aiService.js` na linha que chama o modelo.

**P: Preciso de PostgreSQL agora?**
R: Não, SQLite é suficiente para 100-1000 clientes. Migrar na Fase 5.

**P: Como duplico um cliente?**
R: Mude o `id` e roda `EXEMPLO_CRIAR_CLIENTE.js` novamente com dados diferentes.

**P: Posso usar minha própria chave Gemini?**
R: Sim, coloque em `.env` como `MASTER_GEMINI_KEY`.

**P: Dá pra resetar o contador de um cliente?**
R: Sim, execute: `UPDATE users SET used_messages = 0 WHERE id = 'cliente-id'`
